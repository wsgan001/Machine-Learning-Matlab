To run the code:

Part 1: compare two histograms using the intersection measure

1. Go to image_retrieval_histogram intersection.m 

2. Change the index number of the query imagefile.(It has been assumed for distance calculation that query file will be from the folder)

3. run the code

Part 2: Using earth mover’s distance as similarity measure

1. Go to image_retrieval_emd.m 

2. Change the index number of the query imagefile.

3. run the code
